# Keyboard Shortcuts Reference

## Undo/Redo
| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| **Undo** | `Ctrl + Z` | `Cmd + Z` |
| **Redo** | `Ctrl + Shift + Z` or `Ctrl + Y` | `Cmd + Shift + Z` |

## Canvas Navigation
| Action | Shortcut |
|--------|----------|
| **Select Mode** | `V` |
| **Pan Mode** | `H` or `Space` (hold) |

## Notes
- Keyboard shortcuts are disabled when typing in input fields or text areas
- Both toolbar buttons and keyboard shortcuts work for undo/redo
- Buttons are automatically disabled when no undo/redo is available
- Pressing a new action key after undo clears the redo history

## Supported Operations
Undo/redo tracks changes to:
- ✓ Adding/removing widgets
- ✓ Moving widgets (drag & drop)
- ✓ Resizing widgets
- ✓ Styling widgets (colors, opacity, etc.)
- ✓ Adding/removing tables
- ✓ Adding/removing relationships
- ✓ Adding/removing measures
- ✓ Canvas settings changes
- ✓ Project metadata changes
